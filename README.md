## Game Project Overview 📢

> 도트 레이드는 유니티 엔진을 기반으로 제작된 간단한 액션 장르 게임 입니다.<br>

## Web Project Summary 🚀
<br>
> 웹 게임 Dot Raid 에 등장하는 캐릭터 소개 및 보스 등장 몬스터 들의 소개가 제시 되어 있습니다.
> Dot Raid 관련 웹사이트와 게임 개발진에 대한 소개가 나열 되어 있습니다.
> 게임을 하기위하여 로그인 및 회원가입 페이지도 구성이 되어 있습니다.
